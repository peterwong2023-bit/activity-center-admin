# 活动中心管理后台

Activity Center Admin - 活动营销管理系统后台

## 功能模块

- **仪表板** - 数据概览和统计
- **平台列表** - 平台管理
- **商户列表** - 商户管理
- **活动列表** - 活动配置和管理
- **账号管理** - 用户账号管理
- **日志系统** - 挖矿流水、物品领取、平台日志、审核日志、操作日志

## 技术栈

- HTML5 / CSS3 / JavaScript
- Font Awesome 图标库
- 响应式设计

## 部署

项目已部署到 Vercel，支持静态托管。

## 开发

本地预览：
```bash
python3 -m http.server 8080
```

访问 http://localhost:8080
